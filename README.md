# Мой репозиторий с ДЗ

Короткое описание: небольшой статический проект с примерными HTML-страницами и наборами SVG-изображений (демо-картинки, в т.ч. футбольная галерея).

**Структура проекта**
- `index.html` — главная страница (в корне).
- `page.html`, `index2.html`, `page2.html` — дополнительные примеры страниц.
- `DZ/` — копии страниц и локальная версия в подпапке.
- `images/` — папка с SVG-изображениями, например `logo.svg`, `placeholder.svg`, `football-ball.svg`, `football-field.svg`, `football-trophy.svg`.

**Как посмотреть проект локально**
- Откройте файл `index.html` в браузере двойным кликом.
- В VS Code рекомендую использовать расширение "Live Server" и кнопкой "Go Live" открыть сайт для автоматического обновления при правках.

**Как добавить или заменить изображение**
- Поместите файл в папку `images/` (рекомендуется SVG или PNG).
- Подключите изображение в HTML, например:

```
<img src="images/football-ball.svg" alt="Football ball" width="160" />
```

**Примечания и рекомендации**
- В проекте использованы в основном SVG — они текстовые, легко правятся и масштабируются без потери качества.
- Если нужно PNG, можно сконвертировать SVG в PNG локально (напр., с помощью Python + Pillow или онлайн-конвертеров).
- Файлы, добавленные автоматически: `images/*` содержат демонстрационные картинки (логотип, заглушка, футбольная галерея).

Если хотите, могу: добавить PNG-версии, улучшить подписи, или вставить те же картинки в другие HTML-страницы (например `page.html` или `DZ/index.html`).
